5. Set the same value using innerHTML. What is different, and why?
sol:
Difference:

myName.innerText = "<h2>Hello World</h2>";
Shows the tags as plain text: <h2>Hello World</h2>


myName.innerHTML = "<h2>Hello World</h2>";  Reads the string as HTML and creates an actual <h2> element.
shows: Hello World

Use innerText (or preferably textContent) for normal text. 
Use innerHTML only when you deliberately need to insert HTML, 
especially not with untrusted user input.
